# 电子天平数据处理工具

一个面向 Windows 桌面环境的 Python Excel 处理工具，用于读取电子天平导出的 `.xls` 原始数据，并生成新的 `.xlsm` 结果文件。程序通过 Microsoft Excel COM 自动化处理工作簿，适合需要保留 Excel 公式、图表和宏工作簿格式的实验数据处理场景。

## 功能概览

- 读取电子天平导出的 `.xls` 原始数据文件
- 输出新的 `.xlsm` 结果文件，不覆盖原始数据
- 支持纯水实验和污染物实验两种处理模式
- 提供 Tkinter 图形界面，用于选择文件、输入实验参数和配置处理模式
- 自动生成计算列、结果区域、格式化表格和散点图
- 污染物模式支持自定义平均点散点分析，结果写入 `AA:AB` 区域并生成独立散点图工作表
- 输出文件名自动追加处理模式后缀；同名文件已存在时会追加时间戳

## 运行环境

本工具需要在 Windows 环境中运行，并依赖 Microsoft Excel 桌面版。

建议环境：

- Windows 10 或 Windows 11
- Python 3.10 或更高版本
- Microsoft Excel 桌面版
- `pywin32`

依赖安装：

```powershell
py -m pip install -r requirements.txt
```

如果 `pywin32` 安装后仍无法正常调用 Excel COM，可以执行：

```powershell
py -m pywin32_postinstall -install
```

## 快速开始

在项目目录中运行：

```powershell
py .\process_excel-1.py
```

程序启动后会打开图形界面。按窗口提示选择实验模式、填写参数，并选择电子天平导出的 `.xls` 原始数据文件。

## 输入文件要求

输入文件应满足以下条件：

- 文件扩展名为 `.xls`
- 工作簿中包含名为 `电子天平输出` 的工作表
- 原始数据主要位于 `A:B` 两列
- `A` 列为时间信息
- `B` 列为质量或称量数据

如果输入文件结构不符合要求，程序会在处理时提示错误。

## 使用流程

### 纯水模式

1. 选择“纯水”实验模式。
2. 输入采样时间和膜面积。
3. 选择 `.xls` 原始数据文件。
4. 输入前半段压力和后半段压力。
5. 程序生成纯水计算结果，并另存为新的 `.xlsm` 文件。

### 污染物模式

1. 选择“污染物”实验模式。
2. 输入采样时间和膜面积。
3. 选择 `.xls` 原始数据文件。
4. 输入运行压力。
5. 输入纯水渗透性 `LMH/bar`。
6. 按需配置自定义平均点散点分析。
7. 程序生成污染物计算结果、分析区域和散点图。

自定义平均点散点分析只在污染物模式中生效。取消该设置窗口时，程序会跳过 `AA:AB` 自定义分析区和对应散点图，但仍继续生成污染物模式的基础结果。

## 输出结果

程序会生成新的 `.xlsm` 文件，并保留原始 `.xls` 文件不变。

输出文件名示例：

```text
原文件名-纯水计算后.xlsm
原文件名-污染物计算后.xlsm
```

如果同名输出文件已经存在，程序会自动追加时间戳：

```text
原文件名-纯水计算后-20260510_120000.xlsm
```

## 污染物模式分析结果

污染物模式会生成：

- `G:M` 结果区
- `O:V` 分析区
- 原有散点图工作表
- 可选的 `AA:AB` 自定义平均点分析区
- 可选的自定义平均点散点图工作表

自定义平均点设置规则：

- “每几分钟作为一个平均点”必须是大于 `0` 的数字，可以是整数或小数
- “一共需要几个平均点”必须是大于 `0` 的整数
- 每组点数按 `每几分钟作为一个平均点 * 60 / 采样时间` 计算
- 分钟数必须能被当前采样时间整除
- 原始数据长度必须足够生成用户指定数量的平均点

`AA:AB` 输出含义：

- `AA2`：自定义平均点渗透性标题
- `AB2`：自定义平均点归一化通量标题
- `AA3`：用户输入的纯水渗透性 `LMH/bar`
- `AB3`：归一化基准点
- `AA4` 起：按用户设置分组后计算的渗透性
- `AB4` 起：按 `AA` 列结果归一化后的通量

## 打包为可执行文件

安装 PyInstaller：

```powershell
py -m pip install pyinstaller
```

打包为无控制台窗口的 Windows 可执行文件：

```powershell
py -m PyInstaller --onefile --noconsole .\process_excel-1.py
```

打包完成后，可执行文件会生成在 `dist` 目录中：

```text
dist\process_excel-1.exe
```

目标电脑仍需安装 Microsoft Excel 桌面版，因为程序运行时需要调用 Excel COM。

如果需要查看命令行错误输出，可以去掉 `--noconsole`：

```powershell
py -m PyInstaller --onefile .\process_excel-1.py
```
